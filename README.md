# URL Shortner

A simple URL shortener using NoSQL MongoDB database.


## Tech Stack

- Node.js
- MongoDB
</br></br>

## Run Locally

Clone the project

```bash
  git clone https://github.com/wsasouza/url-shortner-dio
```

Go to the project directory

```bash
  cd url-shortner-dio  
```

Install dependencies

```bash
  yarn 
```

Start the server

```bash
  yarn build:watch
  yarn dev
```

Endpoints:

```bash
  BaseURL: http://localhost:5000 

  /shorten
  Method POST:
  Body: originURL
  

  /:hash
  Method GET:
```
